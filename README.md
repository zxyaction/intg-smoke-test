# Auriko Integrations

Integration plugins for using [Auriko](https://auriko.ai) with third-party tools.

## Available Integrations

| Integration | Tool | Docs |
|-------------|------|------|
| [hermes-agent](./hermes-agent/) | [Hermes Agent](https://hermes-agent.nousresearch.com) | [Guide](https://docs.auriko.ai/integrations/hermes-agent) |

## Getting Started

1. Sign up for an API key at [auriko.ai](https://auriko.ai/signup)
2. Pick an integration above and follow its install instructions

## About

This repo contains integration code only. For the Auriko API itself, see [docs.auriko.ai](https://docs.auriko.ai).
