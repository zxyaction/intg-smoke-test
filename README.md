# Auriko Integrations

[![License: Apache-2.0](https://img.shields.io/badge/License-Apache--2.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)

Integration plugins and adapters for using [Auriko](https://auriko.ai) with third-party tools. Auriko provides a unified API for accessing 120+ LLM models across providers with intelligent routing.

## Available integrations

| Integration | Description | Docs |
|-------------|-------------|------|
| [Hermes Agent](./hermes-agent/) | Provider plugin for Hermes Agent | [Guide](https://docs.auriko.ai/integrations/hermes-agent) |

## Get started

1. [Sign up for an Auriko API key](https://auriko.ai/signup)
2. Follow the install instructions for your tool

## About

This repo contains integration code for connecting third-party tools to the Auriko API. For the API itself, SDKs, and full documentation, see [docs.auriko.ai](https://docs.auriko.ai).
